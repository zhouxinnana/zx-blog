# 欢迎使用 ZX Blog

欢迎来到你的新博客！这是使用 **Nuxt.js** 和 **Vue.js** 构建的一个简单而强大的博客系统。

## 功能特性

这个博客系统包含以下核心功能：

- 📝 **Markdown 支持** - 使用 Markdown 编写文章
- 🏷️ **标签系统** - 为文章添加标签，方便分类
- 📂 **分类管理** - 按分类组织你的文章
- 🔍 **搜索功能** - 快速查找文章
- 📱 **响应式设计** - 完美适配移动设备
- ⚡ **静态生成** - 优化 SEO 和加载性能

## 开始写作

你可以通过以下方式开始写作：

1. 在 `content/posts/` 目录下创建新的 Markdown 文件
2. 使用管理界面创建和编辑文章
3. 添加分类和标签来组织内容
